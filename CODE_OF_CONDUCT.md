# Code of Conduct

## Our Standard

Contributors are expected to communicate respectfully, assume good intent, and keep technical disagreement focused on the work.

Unacceptable behavior includes harassment, discrimination, personal attacks, intimidation, and sustained disruption of discussion spaces.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and other contributions that violate this code of conduct.

## Reporting

Report conduct issues to the repository maintainers through a private GitHub security advisory or direct maintainer contact.
